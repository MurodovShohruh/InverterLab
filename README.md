# InverterLab 🔌

Uch fazali kuchlanish inverteri tahlil dasturi

## Nima qiladi
- Uch fazali signal generatsiyasi (220V, 50Hz)
- PWM modulyatsiya
- Harmonik tahlil (FFT)
- LC Filtr — THD 81% dan 1.4% ga kamaytiradi

## Texnologiyalar
- Python 3.14
- NumPy
- Matplotlib
- SciPy
